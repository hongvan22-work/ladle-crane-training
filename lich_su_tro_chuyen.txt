TẦNG 2 — 4 khái niệm cốt lõi
────────────────────────────────────────

1. BIẾN (Variable)
   Biến là hộp chứa thông tin, có nhãn để tìm lại.

   ten = "Minh"
   tuoi = 25
   luong = 15000000

2. ĐIỀU KIỆN (If/Else)
   Cho máy tính khả năng "nếu... thì... không thì..."

   nếu (số_dư >= số_muốn_rút):
       cho rút tiền
   không thì:
       hiện thông báo "Số dư không đủ"

3. VÒNG LẶP (Loop)
   Làm một việc nhiều lần mà không cần viết lại nhiều lần.

   với mỗi khách_hàng trong danh_sách:
       gửi email "Chúc mừng sinh nhật " + tên_khách

4. HÀM (Function)
   Đóng gói một nhóm lệnh vào một tên, để dùng lại nhiều lần.

   hàm tinh_tien(so_luong, don_gia):
       tong = so_luong * don_gia
       tra_ve tong

════════════════════════════════════════
BUỔI 1 — Biến và Kiểu Dữ Liệu
════════════════════════════════════════
Mục tiêu: Biết cách lưu thông tin vào máy tính và in ra màn hình.

1.1 — In ra màn hình
  print("Xin chào!")
  print("Tôi đang học lập trình")

1.2 — Biến
  ten = "Minh"
  tuoi = 25
  luong = 15000000
  print(ten)   # In ra: Minh

1.3 — Kiểu dữ liệu
  - Số nguyên (int):    5, 100, -3
  - Số thập phân (float): 9.5, 3.14
  - Chữ/Chuỗi (str):   "Minh", "Hà Nội"
  - Đúng/Sai (bool):   True, False

1.4 — Phép tính với số
  a + b   # Cộng
  a - b   # Trừ
  a * b   # Nhân
  a / b   # Chia
  a ** b  # Lũy thừa

1.5 — Ghép chuỗi với f-string
  ten = "Minh"
  tuoi = 25
  print(f"Tên tôi là {ten}, tôi {tuoi} tuổi")

Bài tập:
  1. Tạo biến lưu họ tên, tuổi, thành phố. In ra câu giới thiệu.
  2. Tính tiền hóa đơn: 4 ly cà phê (35k) + 2 bánh mì (15k).
  3. Tính số ngày bạn đã sống (tuổi × 365).

════════════════════════════════════════
BUỔI 2 — Điều Kiện
════════════════════════════════════════
Mục tiêu: Dạy máy tính ra quyết định dựa trên dữ liệu.

2.1 — So sánh
  5 > 3   → True
  5 == 5  → True  (dùng == không phải =)
  5 != 3  → True

2.2 — if / else
  tuoi = 17
  if tuoi >= 18:
      print("Được vào")
  else:
      print("Chưa đủ tuổi")

2.3 — if / elif / else
  diem = 7.5
  if diem >= 9:
      print("Xuất sắc")
  elif diem >= 8:
      print("Giỏi")
  elif diem >= 6.5:
      print("Khá")
  else:
      print("Yếu")

2.4 — Kết hợp điều kiện
  if tuoi >= 18 and co_ve == True:
      print("Được vào rạp")

  if la_hoi_vien or co_ma_giam_gia:
      print("Được giảm giá")

2.5 — Nhận dữ liệu từ người dùng
  ten = input("Bạn tên gì? ")
  tuoi = int(input("Nhập tuổi: "))

Bài tập:
  1. Nhập điểm (0-10), in ra xếp loại.
  2. Nhập số tiền và giá món đồ, in ra "Đủ tiền" hoặc "Thiếu X đồng".
  3. Nhập tháng sinh, in ra mùa tương ứng.

════════════════════════════════════════
BUỔI 3 — Vòng Lặp và Hàm
════════════════════════════════════════
Mục tiêu: Tránh viết lặp code. Tổ chức code thành khối tái sử dụng.

3.1 — Vòng lặp for
  mon_an = ["Phở", "Bún bò", "Cơm tấm"]
  for mon in mon_an:
      print(mon)

  for i in range(5):     # i chạy từ 0 đến 4
      print(f"Lần thứ {i}")

3.2 — Vòng lặp while
  so_tien = 1000000
  while so_tien > 0:
      chi_tieu = int(input("Chi bao nhiêu? "))
      so_tien = so_tien - chi_tieu

3.3 — break và continue
  for i in range(10):
      if i == 5:
          break      # dừng khi i = 5

  for i in range(10):
      if i % 2 == 0:
          continue   # bỏ qua số chẵn

3.4 — Hàm (Function)
  def tinh_tien(don_gia, so_luong):
      tong = don_gia * so_luong
      thue = tong * 0.1
      return tong + thue

  hoa_don_1 = tinh_tien(50000, 3)
  hoa_don_2 = tinh_tien(80000, 2)

Bài tập:
  1. In bảng cửu chương số 7 bằng vòng lặp for.
  2. Viết hàm đổi Celsius → Fahrenheit (F = C × 9/5 + 32).
  3. Viết chương trình đoán số 1-100.

Tóm tắt:
  Buổi 1: =, print(), f"...", input()
  Buổi 2: if, elif, else, ==, and, or
  Buổi 3: for, while, break, def, return

TRƯỚC KHI BẮT ĐẦU — HIỂU ĐÚNG VAI TRÒ CỦA BẠN

Bạn không học để trở thành lập trình viên.
Bạn học để trở thành người biết chỉ đạo Claude làm việc thay mình.

  Lập trình viên:   Tự nghĩ ra giải pháp → Tự viết code → Chạy
  Bạn (với Claude): Mô tả vấn đề → Claude viết code → Bạn chạy và kiểm tra

Kiến thức lập trình bạn cần chỉ đủ để:
  - Hiểu code Claude viết ra đang làm gì (đại ý, không cần từng dòng)
  - Biết sửa những chỗ nhỏ (tên file, cột dữ liệu, con số)
  - Đọc thông báo lỗi và mô tả lại cho Claude
  - Biết cái gì có thể tự động hóa được

Cài đặt một lần:
  1. Python: python.org/downloads
  2. VS Code: code.visualstudio.com
  3. VS Code → cài extension "Python"

────────────────────────────────────────────────────────────────────────────────
BUỔI 1 — Hiểu Cách Máy Tính Làm Việc và Chạy Code Đầu Tiên
────────────────────────────────────────────────────────────────────────────────

PHẦN 1 — Tại sao cần hiểu code dù có Claude?
  Claude không biết file Excel của bạn tên gì, để ở đâu, cột nào là doanh thu.
  Bạn cần hiểu vừa đủ để làm người giám sát thông minh.

PHẦN 2 — Máy tính làm việc như thế nào?
  Thực hiện lệnh theo thứ tự từ trên xuống, chính xác tuyệt đối, không suy luận.

  print("Bước 1: Mở file")
  print("Bước 2: Đọc dữ liệu")
  print("Bước 3: Tính toán")
  print("Bước 4: Lưu kết quả")

PHẦN 3 — Chạy code đầu tiên
  Nhấn nút ▶ (Run) hoặc F5 trong VS Code.

  print("Xin chào!")
  print("Tôi đang học dùng Claude cho công việc văn phòng")
  print(2024 + 1)

PHẦN 4 — Biến
  Biến = hộp chứa thông tin, có đặt tên.
  Cú pháp: tên_biến = giá_trị

  ten_file = "bao_cao_thang_5.xlsx"   # ← bạn cần sửa dòng này
  thu_muc  = "C:/Users/NhanVien/Desktop"

  Khi Claude viết code, phần biến ở đầu file là phần BẠN cần tự sửa.

PHẦN 5 — Kiểu dữ liệu

  Loại           Ví dụ              Dùng khi
  ─────────────────────────────────────────────
  int  (số nguyên)  25, 100, -3      Tuổi, số lượng
  float (thập phân)  9.5, 3.14       Điểm, giá tiền
  str  (chữ)       "Minh", "HN"     Tên, tên file
  bool (đúng/sai)  True, False      Trạng thái

  Quan trọng: 500 (số) khác "500" (chữ).

PHẦN 6 — Cách đặt yêu cầu với Claude

  Template hiệu quả:
  ┌─────────────────────────────────────────────────────────────┐
  │ Viết code Python để [làm gì].                               │
  │                                                             │
  │ Input: [tên file / loại file / file ở đâu]                  │
  │ Cấu trúc dữ liệu: [các cột, các trường thông tin]           │
  │ Output mong muốn: [kết quả trông như thế nào]               │
  │ Điều kiện đặc biệt: [lọc, sắp xếp, bỏ qua gì...]           │
  │                                                             │
  │ Thêm ghi chú tiếng Việt vào code giải thích từng bước.     │
  └─────────────────────────────────────────────────────────────┘

BUỔI 2 — Đọc Hiểu Code và Xử Lý Khi Có Lỗi
────────────────────────────────────────────────────────────────────────────────

PHẦN 1 — Đọc code như đọc công thức nấu ăn

  Khi đọc code, hỏi 3 câu:
    1. Dữ liệu vào là gì?    (Đọc file gì, lấy thông tin gì)
    2. Làm gì với dữ liệu?   (Tính toán, lọc, sắp xếp)
    3. Kết quả ra là gì?     (Lưu file, in ra màn hình)

  Ví dụ đọc code 3 khối:
    ① ĐỌC FILE VÀO:    du_lieu = pd.read_excel("doanh_thu.xlsx")
    ② XỬ LÝ:           ket_qua = du_lieu[du_lieu["Thanh_tien"] > 5000000]
    ③ XUẤT KẾT QUẢ:   ket_qua.to_excel("loc_doanh_thu.xlsx")

PHẦN 2 — Điều kiện (if / elif / else)

  doanh_thu = 75000000

  if doanh_thu > 100000000:
      xep_loai = "Xuất sắc"
  elif doanh_thu > 50000000:
      xep_loai = "Tốt"
  else:
      xep_loai = "Cần cải thiện"

  Khi nào bạn cần sửa? Khi ngưỡng số liệu không đúng thực tế công ty.
  Chỉ đổi con số, không đụng gì khác.

PHẦN 3 — Vòng lặp (for)

  for ten_file in danh_sach_file:
      # xử lý từng file...
      print(f"Đã xử lý: {ten_file}")

  Đọc như văn nói: "Với mỗi file trong danh sách, làm những việc bên dưới."

PHẦN 4 — Đọc và xử lý thông báo lỗi

  Cấu trúc thông báo lỗi:
    File "code.py", line 5    ← lỗi ở dòng nào
    FileNotFoundError: ...    ← lỗi là gì

  3 lỗi phổ biến:
    FileNotFoundError  → Kiểm tra tên file, đường dẫn
    ModuleNotFoundError → Chạy: pip install tên_thư_viện
    KeyError: "Tên_cột" → Kiểm tra tên cột trong Excel có khớp không

  Quy trình xử lý:
    1. Copy toàn bộ thông báo lỗi
    2. Paste vào Claude + mô tả file của bạn
    3. Claude giải thích và sửa

PHẦN 5 — Cài thư viện cần thiết (chỉ cài 1 lần)

  pip install pandas openpyxl python-docx

  Thư viện      Dùng cho
  ──────────────────────────────────────────
  pandas        Xử lý bảng dữ liệu, Excel
  openpyxl      Đọc/viết file Excel (.xlsx)
  python-docx   Đọc/viết file Word (.docx)
  os, shutil    Quản lý file, thư mục
